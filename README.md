# Quoridor
For both touch and mouse devices. Players use drag and drop technique to play the game.

For a detailed look on game rules, see https://quoridor.darkodimitric.com/game_rules.html

To play to game, open https://quoridor.darkodimitric.com

DragDropTouch.js comes from codeproject (https://www.codeproject.com/Articles/1091766/Add-support-for-standard-HTML-Drag-and-Drop-operat).
